#BoraLucrar

O projeto Bora Lucrar é um site de gestão financeira voltado para vendedores ambulantes. 
Ele ajuda a controlar entradas e saídas de dinheiro, calcular lucros e organizar o fluxo de caixa de forma simples e acessível. 
A ideia é facilitar a administração do negócio no dia a dia, permitindo que o vendedor tome decisões melhores e aumente seus ganhos.
