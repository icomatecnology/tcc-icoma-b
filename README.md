# SkillBloom - Website para aprendizado de competências socioemocionais

Bem-vindo ao repositório do projeto **SkillBloom**, uma aplicação web educacional desenvolvida para praticarmos e consolidarmos conhecimentos de **HTML**, **CSS**, **JavaScript**, além de técnicas modernas de **deploy com Firebase Hosting**, **autenticação com Firebase Authentication** e **banco de dados com Firestore**.

---

## 🧭 Objetivo

Criar um jogo interativo que:
- Ajude no desenvolvimento de competências socioemocionais
- Promova autoconhecimento
- Simule situações do dia a dia a dia profissional por meio de um simulador
- Avalie o desempenho do usuário por meio de questionários e pontuação
- Direcione  usuário para melhorias com base nos resultados obtidos
- Incentive a melhoria contínua
---

## 🛠️ Tecnologias Utilizadas

| Tecnologia         | Descrição                                      |
|--------------------|------------------------------------------------|
| HTML5              | Estrutura da aplicação                         |
| CSS                | Estilização responsiva da interface       |
| JavaScript         | Lógica do jogo, questionários, simulador e sistema de pontuação    |

---

## 🚀 Funcionalidades

- [x] Cadastro e login de usuários com Firebase Authentication
- [x] Questionários de avaliação socioemocional
- [x] Simulador de ambiente de trabalho dinâmico
- [x] Sistema de decisões com impacto na pontuação
- [x] Feedback de desempenho com base nas escolhas
- [x] Direcionamneto para a melhoria com base nos resultados

# 👥 Personas de Usuário — SkillBloom

As personas representam os principais usuários da plataforma.

O foco do SkillBloom é ajudar jovens de 16 a 24 anos a desenvolver competências socioemocionais como:

- 🗣 Comunicação
- 💡 Inteligência emocional
- 🤝 Trabalho em equipe
- 👑 Liderança
- ❤️ Empatia

---

| 📚 Escolaridade | Ensino Médio |
| 📍 Localização | Matão – SP |

---

## 🧩 Quem é Lucas?

Lucas está terminando o ensino médio e começa a pensar no futuro profissional.

Mesmo sendo dedicado, possui dificuldade em:

- falar em público;
- trabalhar em grupo;
- controlar a ansiedade.

Ele gosta de tecnologia e prefere experiências práticas e interativas.

---

## 😣 Principais Dores

- Insegurança em apresentações
- Falta de autoconfiança
- Ansiedade social
- Dificuldade de comunicação

---

## 💚 Objetivos

- Melhorar a comunicação
- Desenvolver confiança
- Aprender a controlar emoções

---

## ⭐ Como o SkillBloom ajuda?

O SkillBloom oferece:

- desafios interativos;
- simulações;
- acompanhamento de evolução;
- atividades práticas.
